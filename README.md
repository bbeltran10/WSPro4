# Coffee Shop Website

## Site Description

Coffee Shop is a modern, responsive landing page designed to showcase a local café’s offerings and atmosphere. The site highlights featured menu items, provides location details through an embedded Google Map, and creates a welcoming experience with a clean and visually appealing layout.

The website includes a consistent navigation bar and footer, along with a structured layout that is easy to expand. It is built using semantic HTML and external CSS, making it maintainable and scalable. The design focuses on usability, responsiveness, and a smooth browsing experience across devices.

---

## Pages

1. Home (index.html)

   * Main landing page for the coffee shop
   * Includes a hero section with a welcoming message and call-to-action
   * Displays featured menu items with images and descriptions
   * Contains navigation links to all sections of the page

---

## Sections

1. Navbar

   * Displays the Coffee Shop logo
   * Includes navigation links (Menu, Location, Contact)
   * Features an "Order Online" button

2. Hero Section

   * Eye-catching introduction with headline and tagline
   * Call-to-action button encouraging users to view the menu

3. Menu Section

   * Showcases coffee items such as Espresso, Latte, and Cappuccino
   * Includes images and descriptions for each item
   * Tab layout prepared for future interactivity (Coffee, Tea, Pastries)

4. Location Section

   * Embedded Google Maps iframe
   * Displays the physical location of the coffee shop

5. Footer

   * Contains copyright information
   * Repeats navigation links for easy access

---

## Key Features

* Responsive Design Flexible layout that adapts to different screen sizes
* Clean UI Modern and minimal aesthetic for better user experience
* Embedded Map Integration Google Maps iframe for location display
* Structured Layout Organized sections for easy navigation and scalability
* Expandable Design Ready for additional pages and features
